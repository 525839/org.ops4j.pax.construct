pax:clone

Clones an existing project and produces a script (plus archetypes) to mimic its structure using Pax-Construct 

mvn pax:clone

Mojo Attributes:

  • Requires a Maven 2.0 project to execute.
  • Executes as an aggregator plugin.

Optional Parameters

Name Type Description
includeResources String Comma-separated list of additional non-Maven resources that should be captured.
m_archiverManager ArchiverManager Component factory for various archivers
m_factory ArtifactFactory Component factory for Maven artifacts
m_installer ArtifactInstaller Component for installing Maven artifacts
repair boolean When true, replace any local bundle dependencies with pax-import-bundle commands. Default value is false.

Parameter Details

includeResources

Comma-separated list of additional non-Maven resources that should be captured.

  • Type: java.lang.String
  • Required: No
  • Expression: ${includeResources}
m_archiverManager Component factory for various archivers
  • Type: org.codehaus.plexus.archiver.manager.ArchiverManager
  • Required: No
m_factory Component factory for Maven artifacts
  • Type: org.apache.maven.artifact.factory.ArtifactFactory
  • Required: No
m_installer Component for installing Maven artifacts
  • Type: org.apache.maven.artifact.installer.ArtifactInstaller
  • Required: No
repair When true, replace any local bundle dependencies with pax-import-bundle commands.
  • Type: boolean
  • Required: No
  • Expression: ${repair}
  • Default: false
