Environment variables

PAX_CONSTRUCT_VERSIONforce use of a different plugin version
PAX_CONSTRUCT_OPTIONSlets you set additional Maven plugin options that will apply
to subsequent plugin calls, ie. set it to -o to work offline

General script usage

-hprint basic help text for the command
-oalways overwrite files and entries, instead of warning first
no optionsthe script will prompt for input and suggest default values
